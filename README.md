# ExpenseApp
Uni project - A simple web application to track personal expenses. Users can add, edit, and delete expenses, as well as view a summary of all expenses.  

## Features

- **Single-Page Application (SPA)** built with **React.js**.
- **Backend API** using **Node.js**, **Express**, and **SQLite**.
- **Add/Edit/Delete expenses** with live updates.
- **Expense summary** showing total expenses.
- **Basic input validation** on both client and server sides.
- **Dockerized** for easy deployment with `docker-compose`.
- **Unit and integration tests** for core functionalities.
